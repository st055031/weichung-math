二次函數實驗室

y = a(x - h)² + k
y = 1(x - 0)² + 0
係數 a (開口)1.0
平移 h (左右)0.0
平移 k (上下)0.0
互動觀察重點：
  • 開口方向： 當 a > 0 時開口向上；a < 0 時開口向下。
  • 頂點座標： 拖動 h 與 k，觀察圖上橘色頂點 V(h, k) 的移動。
  • 對稱軸： 虛線代表圖形的對稱軸，其方程式為 x = h。

三次函數實驗室

y = a(x - h)³ + p(x - h) + k
y = 1(x - 0)³ - 3(x - 0) + 0
係數 a (方向)1.0
係數 p (波浪)-3.0
平移 h (左右)0.0
平移 k (上下)0.0
互動觀察重點：
  • 對稱中心： 拖動 h 與 k，觀察圖上紅色中心點 C(h, k) 的移動，整個圖形對此點對稱。
  • 波浪變化： 調整 p 值。當 p 與 a 異號時會有波浪；同號時則為單調遞增/遞減。
  • 中心切線： 橘色虛線為通過對稱中心的切線，其斜率就是 p。
